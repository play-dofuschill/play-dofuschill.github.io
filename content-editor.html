Mode d'emploi

Cet outil lit et réécrit scripts/data/itemDictionary.js, panoplieDictionary.js, familiarDictionary.js et le bloc ZONE_SETS de areaDictionary.js.

  1. Sers le dossier du jeu en local (contexte sécurisé requis par l'API d'écriture) :
    cd play-dofuschill.github.io-main && python3 -m http.server 8000
  2. Ouvre http://localhost:8000/tools/content-editor.html dans Chrome ou Edge.
  3. Clique « Ouvrir le dossier du jeu » et sélectionne la racine du jeu (play-dofuschill.github.io-main). Autorise lecture + écriture.
  4. Édite, puis « Enregistrer » réécrit les fichiers en place.

Astuce : garde Git pour pouvoir annuler. Les images s'affichent via ../images/….
